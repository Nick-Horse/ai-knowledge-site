## 项目简介

为企业定制开发的 AI 智能客服系统，基于大语言模型 + RAG 知识库架构，已深度适配 **Shopify 独立站**场景。

## 核心功能

- **Shopify 深度集成** — 通过 GraphQL Admin API + Webhooks 对接订单、客户、商品数据
- **多轮对话管理** — 上下文感知，自动识别客户意图
- **企业知识库 RAG 检索** — 产品 FAQ、售后政策、物流规则实时查询
- **自动工单创建** — 复杂问题自动生成工单并转接人工
- **多渠道接入** — Web Widget、WhatsApp、Email
- **多语言支持** — 跨境卖家刚需，自动翻译对话
